# 🚗 Smart Parking Management System

A full-stack parking management web application built using Flask and MySQL.

## 🔥 Features

- Secure Admin Login
- Live Parking Slot Monitoring (Available vs Occupied)
- Lot-wise Occupancy Tracking
- Revenue Analytics Dashboard (Daily / Monthly / Yearly)
- Search & Date Range Filtering
- Pagination
- Excel & PDF Report Export
- Dynamic Parking Fee Calculation

## 🛠 Tech Stack

- Python (Flask)
- MySQL
- Pandas
- Bootstrap
- ReportLab

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Setup MySQL database
4. Run:
   python app.py
5. Open:
   http://127.0.0.1:5000
